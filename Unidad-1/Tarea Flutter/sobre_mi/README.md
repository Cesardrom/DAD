# Sobre mí

Esta aplicación Flutter muestra una interfaz personal con información sobre mí, incluyendo una imagen, nombre y textos descriptivos.

## Widgets utilizados

- **MaterialApp**: Raíz de la aplicación con tema Material Design.
- **Scaffold**: Estructura básica con AppBar y body.
- **AppBar**: Barra superior con título y color personalizado.
- **Container**: Contenedor con padding para el body.
- **Column**: Layout vertical para organizar los elementos.
- **ClipOval**: Para hacer la imagen circular.
- **Image.network**: Imagen cargada desde una URL.
- **SizedBox**: Espaciado entre elementos.
- **Text**: Textos con estilos personalizados (tamaño, color, peso).

## Cómo ejecutar

### Prerrequisitos
- Flutter SDK instalado y configurado.
- Un emulador Android/iOS o navegador web.

### Pasos
1. Navega al directorio del proyecto:
   ```
   cd sobre_mi
   ```
2. Instala las dependencias:
   ```
   flutter pub get
   ```
3. Ejecuta la aplicación:
   ```
   flutter run
   ```
