# Tarjeta Mínima

Esta aplicación Flutter muestra una pantalla con una tarjeta mínima que incluye:
- Un icono de avatar (cuenta de usuario).
- El nombre "Juan Pérez".
- Una descripción breve: "Desarrollador móvil".
- Un botón que, al presionarse, muestra un SnackBar con el mensaje "¡Botón pulsado!".

La app está construida con widgets stateless y utiliza Material Design para la interfaz.

## Cómo ejecutar

### Prerrequisitos
- Tener instalado el SDK de Flutter. Si no lo tienes, sigue las instrucciones en [flutter.dev](https://flutter.dev/docs/get-started/install).

### Pasos
1. Navega al directorio del proyecto:
   ```
   cd tarjeta_minima
   ```
2. Instala las dependencias:
   ```
   flutter pub get
   ```
3. Ejecuta la aplicación:
   ```
   flutter run
   ```

Esto iniciará la app en un emulador o dispositivo conectado.
